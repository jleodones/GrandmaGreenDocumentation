    Class SaveManager

    The SaveManager class is in charge of high level saving. It is the only part of the save system that lives in the game. Upon awakening, it instantiates a new SaveController. It also tracks the auto save intervals, then pings the SaveController at appropriate moments. The SaveManager also stores an internal list of every ObjectSaver relevant to the whole game.
